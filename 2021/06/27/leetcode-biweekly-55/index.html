Overview:

  • 题目不难, hard题是数据结构题, 比较麻烦但思维难度不大

  • 最近练习比较少, 状态不太好, 思考与Debug都比较慢.

Solutions:

Remove One Element to Make the Array Strictly Increasing

非常简单的一维dp

Remove All Occurrences of a Substring

  • 为了避免字符串操作的高复杂度, 自己写的简单的双向链表作伪删除, 结果最后发现暴力直接在字符串上操作用时更少, 可能是因为递归函数调用的开销吧, 无语….
  • 避免使用递归函数也没有比暴力更快, 果然medium还是不要想太多…
  • 某个语句放错了位置Debug了半天, 还是不够熟练

暴力:

Maximum Alternating Subsequence Sum

  • 简单的dp, dp[i][j] 表示前i个数里取奇数(j == 1)或偶数(j == 0)个数所能获得的alternating sum
  • 怎么老是不记得要拿上一个状态的结果做保底

Design Movie Rental System

  • 比较麻烦的数据结构题, 但不是很难, 条件卡的也不苛刻
  • C++的set用平衡二叉树实现, 从头开始按顺序遍历很方便
